The website itself is very straight forward. You can search for an anime by name, sort by Title, Episodes, Start Date, Score, and the default search result from the API. You can also toggle by Ascending or Descending.
The bottom of the page has arrows to go to the next page or previous page if your result goes over the limit (20).
All the data queried from the API is saved inside a div and displayed on the website under results. It is controlled by a flexbox.
